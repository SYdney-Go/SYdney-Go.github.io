# 1. 표준 출력 함수
해당 시스템이 가장 기본으로 사용하는 출력 방식 → 모니터

## (1) putchar : 단일 문자 출력 함수 (줄바꿈 X)

``` cpp
#include <stdio.h>

void main()
{
    putchar('H');
    putchar('i');
    putchar('~');
}
```

## (2) puts : 문자열 출력 함수 (줄바꿈 O)

``` cpp
#include <stdio.h>

void main()
{
    puts("Hi~");
}
```
